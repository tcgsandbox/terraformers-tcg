# Terraformers TCG

Two rival gods sculpt a shared world. Claim its lands, command its creatures, and reshape the planet to your will.

A two-player trading card game where **lands** are the core resource. Eight biomes each generate Essence under different conditions — Forests stack, Jungles need units, Mountains reward your spells, Oceans punish your opponent's. Win by controlling 7 lands (**Dominion**) or reducing the opposing god's Divinity to 0 (**Annihilation**).

Play it at [tcg-sandbox.com](https://tcg-sandbox.com).

## About this repo

Terraformers TCG was built as the worked example for the [Use AI to Create a Game](https://tcg-sandbox.com/docs/guides/use-ai-to-create-a-game) guide on tcg-sandbox.com. Every part of it — rules, cards, lore, board layout — was created by having a conversation with Claude Code inside the [tcg-sandbox terraform template](https://github.com/tcgsandbox/terraform-template) and letting it write the configuration.

If you want to build your own game the same way, start with the guide.
